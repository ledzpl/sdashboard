# KOSPI 실시간 주식 정보 대시보드

`AGENTS.MD` 요구사항을 기반으로 구현된 KOSPI 전용 실시간 패널 대시보드입니다.

## 포함된 패널

- 실시간 호가 및 당일 고가/저가
- 실시간 가격 차트 (5초 시세 누적)
- 5단계 호가/잔량
- 수급 동향 (외국인/기관/개인)
- 종목 기본 금융 정보
- 한국 주식 시장 기본 정보(KRX 세션)
- 한국 마켓 전체 상황(코스피/코스닥/환율/글로벌 변수)
- 한국 주요 뉴스
- 해당 종목 주요 뉴스
- 한국 시장 영향 해외 뉴스
- 실시간 TV 방송(동영상)

## 추가 기능

- 종목명 자동완성 검색
  - 6자리 종목코드 없이 `삼성전자`, `현대차`처럼 이름으로 바로 조회 가능
  - KOSPI 전체 종목 목록을 실시간으로 수집해 자동완성 후보 제공
- 패널 레이아웃 저장
  - 패널 드래그 앤 드롭 재배치
  - 패널 펼침/접힘 상태 저장
  - 브라우저 재접속 후 레이아웃 자동 복원
  - `레이아웃 초기화` 버튼 제공

## 기술 구성

- 백엔드: Node.js 내장 HTTP 서버 (`server.mjs`)
- 프론트엔드: Vanilla JS + CSS 패널 UI (`public/app.js`, `public/styles.css`)
- 데이터 소스:
  - 네이버 모바일 증권 API (국내 종목/호가/국내 지수)
  - 네이버 모바일 `front-api/search` (종목 자동완성 검색)
  - STOOQ (환율/해외 지수/원자재 보조 지표)
  - Yahoo Finance (종목명 입력 시 심볼 해석 및 폴백)
  - Google News RSS (국내/종목/해외 뉴스)
  - YouTube Live embed (경제 방송 라이브 채널)

## 실행 방법

```bash
npm start
```

개발 모드(파일 변경 감시):

```bash
npm run dev
```

기본 주소: [http://localhost:3000](http://localhost:3000)

`npm run dev`는 `0.0.0.0`으로 바인딩되므로 같은 네트워크의 다른 기기에서는 `http://<개발PC_IP>:3000`으로 접속할 수 있습니다.

## 구조

- `/server.mjs`: 실시간 데이터 API 프록시 및 정적 파일 서빙
- `/public/index.html`: 대시보드 레이아웃
- `/public/app.js`: 데이터 소스 폴링/패널 렌더링
- `/public/styles.css`: 가독성 중심 패널 UI 스타일

## 확장 포인트

패널 추가는 `public/app.js`에서 아래 두 영역만 확장하면 됩니다.

1. `dataSources`: 새로운 실시간 API 소스 정의
2. `panels`: 패널 메타/렌더 함수 등록
